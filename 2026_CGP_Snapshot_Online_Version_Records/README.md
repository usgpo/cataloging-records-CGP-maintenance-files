# 2026 CGP Snapshot Online Version Records

There are 223,235 records in this collection. Metadata specifications to compile the record set are based on the MARC [008 Fixed-Length Data Elements](https://www.loc.gov/marc/bibliographic/bd008.html) and the [856 Electronic Location and Access](https://www.oclc.org/bibformats/en/8xx/856.html) fields:

- 008/23 or 008/29 must be encoded "o"
- 856 40 must have a PURL with address prefix of <https://purl.fdlp.gov/GPO/>
